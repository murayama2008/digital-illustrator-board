Aplikasi Digital Illustrator Board sederhana

Aplikasi menggambar sederhana memakai Python Tkinter

## Fitoer
- Menggambar dengan gerakan jari (masuk ke mode drawing jika jempol dan telunjuk di tempelkan)
- Pilih warna
- Clear canvas
- Save ke format .png

## Requirements
- Python 3.9 - 3.11
- Webcam

Install dependency:

pip install opencv-python mediapipe customtkinter pillow numpy

Jalankan program:

python air_draw.py
